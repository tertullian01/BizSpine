Install BizSpine (no SSH required)

This guide is for non-technical users installing from the release ZIP on Apache shared hosting (cPanel, Plesk, etc.).

You need: PHP 8.0+, SQLite, and a folder on your site (default name BizSpine).

1. Upload the ZIP

  1. In cPanel, open File Manager.
  2. Go to your home folder (one level above public_html).
  3. Upload BizSpine-release.zip and Extract it.

After extraction you should have:

  • bizspine-backend/ — next to public_html (not inside it)
  • public_html/BizSpine/ — your storefront and installer

2. Run the web installer

  1. Open https://YOUR-DOMAIN.com/BizSpine/install.php in your browser.
  2. Enter your store URL, store name, and administrator account (or tick Load demo data for a ready-made sample store).
  3. Click Install BizSpine.

3. After installation

  • Visit your store at https://YOUR-DOMAIN.com/BizSpine/
  • Admin dashboard: https://YOUR-DOMAIN.com/BizSpine/admin
  • Delete install.php from public_html/BizSpine/ for security.

Demo accounts (if you loaded demo data)

Password for all demo users: Example123!

Different folder name?

If your folder is not BizSpine, edit RewriteBase in public_html/YourFolder/.htaccess and api/.htaccess to match (e.g. /YourFolder/).
